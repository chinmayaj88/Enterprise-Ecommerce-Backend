## Development Process

1. **Fork** the service repository you want to contribute to
2. **Clone** your fork
3. **Create** a feature branch
4. **Make** your changes
5. **Test** your changes
6. **Submit** a pull request in that service repository

See [Development Workflow](./02-development-workflow.md) for details.

## Service-Specific Contribution

### Choose a Service

Pick the service you want to contribute to:
- Gateway Service - API gateway, routing
- Auth Service - Authentication, authorization
- User Service - User management
- Product Service - Product catalog
- Cart Service - Shopping cart
- Order Service - Order processing
- Payment Service - Payment processing
- Notification Service - Notifications
- Discount Service - Coupons, promotions
- Shipping Service - Shipping, tracking
- Return Service - Returns, refunds
